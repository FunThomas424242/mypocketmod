MyPocketmod - Benutzerhandbuch

Table of Contents
  • Benutzung

Kommandozeilenprogramm welches 8 PDF Dokumente in eine PDF Seite konvertiert, die im Pocketmod Style gefaltet werden kann. So werden die Seiten:

zu folgender Seite zusammengefügt:

Figure 1. MyPocketmod.pdf

Benutzung

  1. Konsole öffnen und ins "Projektverzeichnis" wechseln.

  2. Im Projektverzeichnis 8 PDF Dokumente ablegen, welche bitte Seite1.pdf, Seite2.pdf bis Seite8.pdf benannt sind.

  3. Auf der Kommandozeile java -jar <pfad zum JAR Archiv> eingeben und mit Enter zur Ausführung bringen → Es entsteht eine Konfigurationsdatei und eine PDF Datei welche das final Pocketmod enthält.

Beispiel einer Unix Konfigurationsdatei
#Configuration für mypocketmod
#Sun Jun 09 21:15:20 CEST 2019
mypocketmod.sourcedir=src/test/resources/parts/
mypocketmod.targetdir=generated/src/test/
mypocketmod.output.filename=MyPocketmod.pdf
mypocketmod.page1.filename=TODOListe.pdf
mypocketmod.page1.orientation=auto
mypocketmod.page2.filename=CodeReview.pdf
mypocketmod.page2.orientation=auto
mypocketmod.page3.filename=CodeSmells.pdf
mypocketmod.page3.orientation=hochformat
mypocketmod.page4.filename=Termine.pdf
mypocketmod.page4.orientation=auto
mypocketmod.page5.filename=EssenTrinken.pdf
mypocketmod.page5.orientation=auto
mypocketmod.page6.filename=Notizen.pdf
mypocketmod.page6.orientation=auto
mypocketmod.page7.filename=Ausgaben.pdf
mypocketmod.page7.orientation=auto
mypocketmod.page8.filename=SQLZugriff.pdf
mypocketmod.page8.orientation=querformat
